# Enterprise Project Case Studies

This repository documents selected enterprise projects I contributed to during my time at **Beximco Pharmaceuticals Ltd.**

The original source code for these projects is **not published publicly** because it was developed in a professional company environment and is subject to confidentiality and ownership restrictions.

Instead of sharing private company code, this repository presents:
- project context
- business problem
- users and use cases
- my role and contributions
- technology stack
- key features
- engineering lessons learned

## Why this repository exists

Many strong professional projects cannot be shown as public source code. However, they still represent real software engineering experience, real users, real workflows, and real delivery work.

This repository is intended to provide a clear and honest view of that experience.
